# RECIPE APP
This is a recipe app made using Flask, Python & SQLite for University.

## Features

- Create Account ✅
- Login ✅
- Create Recipe ✅
- View Recipe ✅
- Delete Recipe ⏳
- Comment on recipies ⏳
- Add images to recipies ⏳
- Search recipies ✅
- Show users recipies ✅
- Add/Show recipe ratings ⏳
- Show top recipes ⏳


## Build instructions
1. Clone the repo
2. `$ pip install flask`
3. `$ sqlite3 database.db < schema.sql`
3. `$ flask run`
