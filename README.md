# RECIPE APP
This is a recipe app made using Flask, Python & SQLite for University.

## Features
- Create Account
- Login
- Create Recipe
- View Recipe
- Delete Recipe
- Comment on recipies
- Add images to recipies
- Search recipies
- Show users recipies
- Add/Show recipe ratings


## Build instructions
1. Clone the repo
2. Run<br/>`$ pip install flask`
3. Run<br/>`$ flask run`
